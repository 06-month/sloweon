# 페르소나 서브 에이전트

이 디렉터리는 남성 컨템포러리 패션 쇼핑몰의 고객 페르소나 기반 서브 에이전트 정의를 보관한다.

## 사용 원칙

- 디자인, 개발, QA, 런칭 판단 시 최소 3개 페르소나의 관점으로 검토한다.
- 각 에이전트는 `docs/customer-personas.md`의 페르소나 정의를 기준으로 피드백한다.
- 에이전트 피드백은 개인 취향이 아니라 구매 가능성, 이해 가능성, 신뢰, 사이즈 확신, 전환 흐름을 기준으로 판단한다.
- 페르소나 간 의견이 충돌하면 `docs/shopping-mall-pyd.md`의 P0 요구사항과 `plan.md`의 검증 체크리스트를 우선한다.

## 에이전트 목록

- `persona-kim-dohyun.md`: 스타일 입문 남성
- `persona-park-junseo.md`: 데일리 직장인
- `persona-lee-taeo.md`: 트렌드 얼리어답터
- `persona-choi-minjae.md`: 사이즈 민감 재구매 고객
