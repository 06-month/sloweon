## 1. 브랜드별 디자인 특징

### 1) 벨리에 (BÉLIER) — belier.co.kr
**디자인 특징**
- 세리프 워드마크(BÉLIER, 악센트 강조)로 유럽 브랜드 무드
- 상단에 "가입만으로, 웰컴 쿠폰팩" **마퀴(흐르는) 티커**로 혜택 상시 노출
- 헤더에 **Seoul, Korea / 실시간 시각 / 날씨(30°C) 위젯** → 감성 디테일
- 풀블리드 시즌 캠페인 히어로(S/S 2026, "정교한 설계, 그 틈에서 발견한 낭만")
- **이미지 위 카피 오버레이형 카테고리 카드**(남성/여성 각각: 스웻&티셔츠·데님·악세서리 등)
- **화면 하단 고정형 다크 pill 네비게이션**(검색 + 남성복/여성복/아카이브/저널/스토리 + 문의하기)
- 저널·아카이브 콘텐츠(인터뷰/에세이) 비중이 커서 매거진 성격이 강함

### 2) 아트이프액츠 (Art if acts) — artifacts.co.kr
**디자인 특징**
- **흑백 필름 사진** 풀블리드 히어로 → 아날로그·아카이브 무드
- 세리프 로고 중앙, 상단 좌측(ABOUT/COLLECTION/STOCKIST/COMMUNITY)·우측(LOGIN/MY PAGE/CART/검색)
- 서브 상품 네비(NEW/26SU/26SS/Outer/Top/Bottom/Acc/Sale)가 이미지 위에 겹쳐 배치
- 홈은 사실상 **풀스크린 이미지 캐러셀 → 곧바로 푸터**(상품·가격·CTA 없음)
- 채널톡 챗 상담 위젯, 우하단 "A" 플로팅 로고
- 톤: 담담·모노크롬, 편집숍/룩북형

### 3) 포터리 (POTTERY) — ptry.co.kr
**디자인 특징**
- 볼드 산세리프 로고, 헤더에 검색/로그인/장바구니 + **한국어/EN 토글**
- **남성/여성 좌우 분할 히어로**, 이후 카테고리별(남성 셔츠/여성 셔츠 등) 분할 카드가 반복
- 각 카드에 **스토리텔링 카피 + "지금 쇼핑하기 / 더 알아보기" 명확한 CTA**
- 진입 시 **"신규 회원 10% 혜택 받기" 멤버십 팝업 모달**(MEMBERS ONLY)
- 1:1 Talk 챗 위젯
- 톤: 무채색 그레이의 정제된 "유니폼/미니멀" 감성

### 4) 블랭크룸 (BLANKROOM) — blankroom.house
**디자인 특징**
- **극단적 미니멀·비대칭 스플릿**(좌측 이미지 스택, 우측 대량 여백)
- 초소형 타이포(blankroom.house / Man / her / [ Discover ])로 아트북·갤러리 무드
- 시즌 라벨만 얹은 정적 레이아웃(26 Summer / 26 Summer·Resort)
- **푸시 알림 스타일 멤버십 토스트 팝업**("BLANKROOM 방금 · 신규 5% 쿠폰 + 3,000 포인트 · Member Join") → 차별화 포인트
- 원형 플로팅 브랜드 버튼
- 톤: 화이트 스페이스 극대화, 미술관 같은 감성

### 5) 인사일런스 (INSILENCE) — insilence.co.kr
**디자인 특징**
- 자연광 필름 감성 풀블리드 히어로("26HS - MEN / New Arrivals" 캡션 CTA)
- **관습적이지만 안정적인 상단 네비**: 좌측(남성복/여성복/SPECIAL OFFER[레드]/컬렉션/컨텐츠/매장안내/검색), 우측(언어/고객센터/계정/쇼핑백/멤버십)
- **홈에 실제 상품 그리드**(MEN BEST / WOMEN BEST 4열, 상품명 + 정가 취소선 + 할인가 + 할인율 레드 표기)
- 챗 위젯, 상단으로 이동 버튼
- 톤: 에디토리얼 + 커머스의 균형이 가장 좋음

### 6) 쿠어 (COOR) — coor.kr
**디자인 특징**
- 6곳 중 유일하게 **컬러감이 살아있는 필름 캠페인 히어로**(장미정원 스타일링)
- 세리프 소문자 로고 "coor" + **넓은 자간의 중앙 정렬 네비**(shop/campaign/lookbook/stores/help/login/cart)
- 홈은 **단일 캠페인 이미지 → 푸터**, 푸터에 **초대형 세리프 "coor" 워드마크**로 타이포그래피 임팩트
- 팝업·챗 없음 → 매우 정적·정갈
- 톤: 유럽 하이패션 매거진, 타이포 중심

## 2. 브랜드별 장단점

### 벨리에
- **장점**: 하단 고정 네비바로 어디서든 탐색 가능 / 날씨·시각 위젯 등 감성 디테일 / 저널·아카이브로 브랜드 서사 풍부 / 상단 쿠폰 티커로 혜택 즉시 인지
- **단점**: 하단 고정바가 콘텐츠 하단을 가림(특히 모바일) / 이미지 위 텍스트 오버레이 가독성 편차 / 남·여 혼재 구성으로 "남성 브랜드" 포커스가 흐려짐 / 홈에서 가격 확인 불가

### 아트이프액츠
- **장점**: 강한 무드·아이덴티티(흑백 필름) / 군더더기 없는 몰입형 비주얼
- **단점**: 홈에 상품·가격·CTA가 없어 **구매 동선이 약함** / 로고·네비가 이미지 위에 겹쳐 가독성 저하 / 색보정 이슈로 실물 색과 차이(리뷰 지적) / 첫 화면에서 "무엇을 파는지·어디를 누를지" 불명확

### 포터리
- **장점**: CTA·정보 구조가 명확 / 남·여 균형 / 한/영 토글로 글로벌 대응 / 카피 스토리텔링으로 상품 가치 전달
- **단점**: 진입 팝업이 콘텐츠를 가리고 **스크롤해도 닫힘이 불편**(닫기 버튼 작음) / 분할 카드 반복으로 레이아웃이 다소 단조 / 히어로에서 실제 상품·가격이 바로 안 보임

### 블랭크룸
- **장점**: 압도적 차별성·아이덴티티 / 세련된 여백 운용 / 푸시 알림형 팝업 등 위트
- **단점**: 초소형 폰트로 가독성·발견성 저하 / 내비게이션이 직관적이지 않아 러닝커브 / 상품 접근성이 매우 약함 / 대량 여백이 모바일에서 빈 화면처럼 보일 위험

### 인사일런스
- **장점**: **가장 실용적**(가격·할인율·상품을 홈에서 즉시 노출) / 명확한 정보 위계 / 다국어(KR/EN/JP) / SPECIAL OFFER 레드 강조로 프로모션 인지 용이
- **단점**: 정보 밀도가 높고 다소 관습적이라 **비주얼 차별성은 낮음** / 헤더 항목이 많아 첫인상이 복잡 / "감성"보다 "쇼핑몰" 느낌이 강함

### 쿠어
- **장점**: 정갈하고 고급스러운 타이포그래피 / 방해요소(팝업·챗) 없이 깔끔 / 넓은 자간 네비로 상단 구조가 명확
- **단점**: 홈에 상품·가격이 전혀 없어 구매 동선이 약함 / 정보량 부족(첫 방문자에게 불친절) / 인터랙션 최소로 재방문 유인이 약함

## 3. 공통 문제점 (업계 전반)

1. **"예쁨"에 치우쳐 커머스 기본기가 약함** — 절반이 홈에서 가격/상품/장바구니 동선을 숨김. 첫 방문자가 "뭘 파는지, 얼마인지"를 3초 안에 못 잡음.
2. **이미지 위 텍스트 오버레이 남발** — 배경 이미지에 따라 로고·메뉴·카피 가독성이 들쭉날쭉(아트이프액츠·벨리에).
3. **팝업 UX 마찰** — 진입 즉시 뜨는 쿠폰/멤버십 모달의 닫기 버튼이 작거나, 스크롤해도 남아 콘텐츠를 가림(포터리).
4. **모바일 배려 부족 위험** — 초소형 타이포(블랭크룸), 하단 고정바(벨리에), 대량 여백은 모바일에서 역효과 가능.
5. **검색·필터·사이즈 가이드의 존재감 약함** — 감성 우선이라 실제 구매 결정 도구가 뒤로 밀림.
6. **접근성(A11y) 취약** — 저대비 텍스트, 얇은 폰트, 이미지 대체텍스트/포커스 상태 미흡 가능성.

## 4. "더 유저친화적 + 더 예쁘게" 개선안 (실행 체크리스트)

### A. 첫 화면(히어로)에서 감성과 상업성 동시 달성
- [ ] 풀스크린 캠페인은 유지하되, **하단에 "베스트 4~6종 상품 스트립"**(썸네일+상품명+가격+할인율)을 항상 노출 → 인사일런스식 커머스 + 쿠어식 감성 결합
- [ ] 히어로에 **명확한 1차 CTA 1개**("SHOP 26SS" 등)만 크게, 부차 링크는 작게(정보 위계 정리)
- [ ] 캠페인 위 텍스트는 **반투명 그라디언트 오버레이 or 세이프 존**을 두어 이미지 무관하게 가독성 확보

### B. 내비게이션 & 정보 구조
- [ ] 상단 네비는 **최대 5~6개 대분류**로 압축(남성/여성/컬렉션/저널/스토어). 나머지는 메가메뉴로
- [ ] 검색을 **아이콘+플레이스홀더가 보이는 상시 바** 또는 상단 고정으로 승격, 인기검색어·최근 본 상품 노출
- [ ] 모바일: 하단 **탭바(홈/카테고리/검색/위시/마이)** 도입하되 콘텐츠를 가리지 않게 반투명·자동 숨김 처리
- [ ] 브레드크럼 + "최근 본 상품" 레일로 회유 동선 강화

### C. 팝업/온보딩 마찰 제거
- [ ] 진입 팝업은 **1회 노출 + 3~5초 지연 + 큰 닫기(X) + 스크롤 시 자동 축소(코너 토스트)** → 블랭크룸의 "푸시 알림형 토스트"가 좋은 레퍼런스
- [ ] 혜택은 팝업 대신 **상단 슬림 배너(닫기 기억)**로 상시 안내

### D. 상품 탐색·구매 전환(핵심)
- [ ] PLP(목록): **호버 시 착용컷↔디테일컷 전환**, 사이즈/컬러 스와치, 품절/재입고 뱃지, 빠른 담기
- [ ] 필터/정렬: 가격·사이즈·컬러·핏·소재 **다중 필터 + 결과 수 실시간** 표기
- [ ] PDP(상세): **사이즈 가이드(실측표+모델 착용 스펙)**, 리뷰(핏/키/몸무게 태그), 코디 추천, 배송/교환 요약을 상단에 고정
- [ ] 장바구니·위시는 **낙관적 UI(즉시 반영)** + 미니 카트 슬라이드

### E. 비주얼 디테일로 "더 예쁘게"
